> 如你所见，这就是一个非常简单的音乐播放器，实现了本地音乐播放器的我和另外一个使用IMP的朋友大部分能用上的功能，部分功能未经验证其高可靠性。

# 部署？
```cmd
pnpm/npm install

# 使用Tauri dev环境
npm run tauri dev

#使用Vite dev环境
npm run dev
```

# 打包
```
npm run tauri build
```

# 注意
程序会在运行的目录下生成一个配置文件夹用于记录配置
